О приложении

Версия 2.0.0

TaskFlow

Простое и быстрое приложение для управления задачами. Работает офлайн благодаря технологии Service Worker и кэшированию (App Shell).

Возможности

  • ✓ Работа без интернета (PWA / App Shell)
  • ✓ Синхронизация задач в реальном времени (WebSocket)
  • ✓ Push-уведомления о новых задачах
  • ✓ Хранение данных в localStorage
  • ✓ Установка на рабочий стол (installable PWA)

Технологии

  • HTML5 / CSS3 / JavaScript
  • Service Worker + Cache API
  • Socket.IO (WebSocket)
  • Web Push API + VAPID
  • Node.js + Express
